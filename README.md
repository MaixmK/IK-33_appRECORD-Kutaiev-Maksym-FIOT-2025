# Онлайн-каталог комп’ютерних комплектуючих

Веб-застосунок для перегляду та підбору комп’ютерних комплектуючих з підтримкою
фільтрації, сортування, пагінації, кошика покупок та авторизації користувачів.
Проєкт реалізовано у навчальних цілях з використанням HTML, CSS та JavaScript
без серверної частини.

---

## Функціональні можливості

- перегляд каталогу комп’ютерних комплектуючих;
- фільтрація та пошук товарів;
- сортування за ціною та рейтингом;
- пагінація каталогу;
- перегляд детальної інформації про товар у модальному вікні;
- додавання товарів у кошик;
- зміна кількості товарів та видалення позицій;
- підрахунок загальної вартості замовлення;
- збереження кошика між сесіями за допомогою LocalStorage;
- авторизація та реєстрація користувачів (клієнтська логіка);
- синхронізація кошика між вкладками браузера;
- адаптивний інтерфейс (десктоп, планшет, смартфон);
- додатковий інтерактивний модуль — калькулятор.

---

## Використані технології

- HTML5 — структурна розмітка сторінок
- CSS3 (Flexbox, Grid, Media Queries) — стилізація та адаптивність
- JavaScript (ES6+) — програмна логіка
- DOM API — динамічна робота з елементами сторінки
- LocalStorage API — збереження даних користувача
- Git / GitHub — система контролю версій

## Запуск проєкту

### Локальний запуск
1. Клонувати репозиторій:
   git clone https://github.com/MaixmK/IK-33_appWEB-Kutaiev-Maksym-FIOT-2025.git
2. Відкрити файл index.html у браузері.

### Рекомендований спосіб
Для коректної роботи JavaScript та LocalStorage рекомендується запускати проєкт
через локальний веб-сервер (наприклад, Live Server у Visual Studio Code).
